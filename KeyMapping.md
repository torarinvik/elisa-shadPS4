# shadPS4 Default Keyboard Mapping

These are the default keyboard bindings from shadPS4's generated input config.
Click or focus the game window before pressing keys.

| PS4 control | Keyboard |
|---|---|
| Cross | `X` or numpad `2` |
| Circle | `C` or numpad `6` |
| Square | `Z` or numpad `4` |
| Triangle | `V` or numpad `8` |
| Options | `Enter` |
| Touchpad center | `Space` |
| D-pad up | `Up Arrow` |
| D-pad down | `Down Arrow` |
| D-pad left | `Left Arrow` |
| D-pad right | `Right Arrow` |
| L1 | `Q` |
| R1 | `U` |
| L2 | `E` |
| R2 | `O` |
| L3 | `N` |
| R3 | `M` |
| Left stick left | `A` |
| Left stick right | `D` |
| Left stick up | `W` |
| Left stick down | `S` |
| Right stick left | `J` |
| Right stick right | `L` |
| Right stick up | `I` |
| Right stick down | `K` |

## Useful Emulator Hotkeys

These come from the default global input config.

| Action | Keyboard |
|---|---|
| Show FPS / debug overlay | `F10` |
| Pause | `F9` |
| Reload inputs | `F8` |
| Toggle fullscreen | `F11` |
| Capture frame / RenderDoc hotkey | `F12` |
| Screenshot with overlays | `Left Alt` + `F12` |
| Toggle mouse to joystick | `F7` |
| Toggle mouse to gyro | `F6` |
| Toggle mouse to touchpad | `Delete` |
| Add virtual user | `F5` |
| Remove virtual user | `F4` |
| Open emulator settings | `F3` |
| Quit hotkey | `Left Ctrl` + `Left Shift` + `End` |
| Volume up | numpad `+` |
| Volume down | numpad `-` |

## Notes

- Game-specific mappings may override these defaults in `~/Library/Application Support/shadPS4/input_config/<TITLE_ID>.ini`.
- The shared default mapping is generated at `~/Library/Application Support/shadPS4/input_config/default.ini`.
- If a game shows `Press` with a blank icon, try `Enter`, `Space`, `X`, and numpad `2` first.
